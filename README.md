# DAILY TASK — Python Constructor Tutorial

This repository contains a single Python tutorial script demonstrating class constructors (`__init__`) and object initialization in Python.

## Included Script

- `PYTHON CONSTRUCTORS (__init__) #           BASIC TO ADVANCED.py`
  - Explains constructor basics and parameterized constructors
  - Shows default values, multiple objects, nested object initialization, and class variables
  - Includes real-world examples like bank accounts, student result calculations, and device objects
  - Demonstrates constructor behavior, pseudo-overloading, and destructor usage (`__del__`)

## What You'll Learn

- How constructors initialize object data automatically
- How to pass parameters into `__init__`
- How to use default constructor values and class variables
- How to create nested objects and object lists
- How to implement simple real-world object models
- How destructors can clean up objects when deleted

## Getting Started

### Prerequisites

- Python 3.8 or newer
- No external dependencies required

### Run the Script

```bash
python "PYTHON CONSTRUCTORS (__init__) #           BASIC TO ADVANCED.py"
```

## Notes

- This script is designed for learners who are new to Python OOP and constructors.
- The examples are printed in clearly labeled sections to make the flow easy to follow.
- You can extend the script with inheritance, encapsulation, and polymorphism exercises.
